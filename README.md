# 🏀 Real-Time Sports Betting Intelligence Platform

A real-time data engineering project that streams live sports odds and prediction market data to surface AI-generated sharp money signals and market divergences.

## Tech Stack
- **Ingestion:** The Odds API + Kalshi API → Apache Kafka (AWS MSK)
- **Transport:** Amazon Kinesis Data Firehose → S3
- **Processing:** Databricks Structured Streaming + ML (Isolation Forest)
- **Warehouse:** Snowflake
- **Orchestration:** Apache Airflow (AWS MWAA)
- **Security:** AWS Secrets Manager, CloudTrail, IAM, KMS
- **Dashboard:** Streamlit (public)
